| Модуль            | Описание                    | Основной функционал                                                                                                                      |
|:------------------|:----------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------|
| **`Application`** | Точка входа в приложение    | Запуск сервера, инициализация Flask инстанса, подключение логгера и связывание всех модулей.                                             |
| **`Config`**      | Файл конфигурации           | Чтение переменных из `.env`, настройки `SQLALCHEMY_DATABASE_URI`, ключи `SECRET_KEY`, `TURNSTILE_SECRET` и параметры Telegram.           |
| **`DB Models`**   | Схемы базы данных           | Описание моделей `User` и `Message` через SQLAlchemy, настройка связей между таблицами.                                                  |
| **`Auth`**        | Система авторизации         | Логика эндпоинтов `/login`, `/api/register`, `/logout`. Хэширование паролей, проверка капчи Turnstile, управление Flask сессиями.        |
| **`Chat`**        | Логика мессенджера          | Эндпоинты интерфейса `/chat` и API `/api/messages` (GET/POST). Защита от XSS-уязвимостей (`markupsafe.escape`).                          |
| **`Telegram`**    | Интеграция с TG             | Изолированная работа с `Telethon`. Полноценное внедрение интеграции чатов из Telegram в Tornado.                                         |
| **`Utils`**       | Вспомогательные инструменты | Функция `get_randomization`, парсинг расписания групп (`get_groups`), получение IP сервера, проверка лицензионного ключа (`verify_key`). |
| **`System`**      | Системные эндпоинты         | Маршруты `/ping` и `/api/ping`.                                                                                                          |
| **`Logger`**      | Логирование событий         | Логирование событий, происходящих во время выполнения кода в файл `logs/log-DATE.log` или просто в консоль.                              |
