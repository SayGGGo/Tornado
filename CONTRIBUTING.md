## Вы можете  помочь развитию проекта!
> **Внимание!** Проект итоговый, поэтому, к сожалению, ваши изменения не будут представленные на презентации.

## Как внести вклад
1. Сделайте Fork репозитория.
2. Создайте ветку для вашей функции (`git checkout -b feature/NewFunc`).
3. Зафиксируйте изменения (`git commit -m 'Add some NewFunc'`).
4. Отправьте ветку в ваш репозиторий (`git push origin feature/NewFunc`).
5. Откройте Pull Request.

## Требования к коду
- **SOLID**: Обязательно
- **Стек**: Основной язык — Python 3, веб-фреймворк — Flask.

## Оформление коммитов
- Используйте понятные описания на английском языке
- Указывайте номер Issue, если коммит решает конкретную проблему.
